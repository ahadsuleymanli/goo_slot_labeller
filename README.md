What you need:
- Python 3.6
- pip for Python 3.6
- pipenv (pip install --user pipenv)

How to run:
1. clone the repository
--- in the project root directory:
2. pipenv install
3. pipenv shell
--- in ./mysite directory:
4. python manage.py runserver
5. open http://127.0.0.1:8000/index/ in your browser
